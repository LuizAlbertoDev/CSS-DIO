Agrupamento de Seletores

Podemos apolicar as mesmas regras CSS para varios seletores diferentes. aPara isso, podemos separar os seletores utilizando uma virgula(,)

Assim, nao precisamos ficar copiando e colando os mesmo codigos para todos, nos podemos fazer um so bloco de CSS e teutilizar as mesmas regras!
